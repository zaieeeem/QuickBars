# Bing-Bong — Privacy Policy

Bing-Bong is an open-source Android TV / Android launcher and quick-controls app for Home Assistant. It is a client for a Home Assistant server that **you** own and operate.

## What data is collected
**None.** Bing-Bong has no analytics, no advertising, no crash-reporting service, and no backend server of its own. The developer collects, stores, and receives no personal data.

## Data you provide to your own server
To function, Bing-Bong stores — **only on your device** — the URL of your Home Assistant instance and a long-lived access token that you generate. These are used solely to communicate directly with your own Home Assistant server. They are never transmitted to the developer or any third party.

## Permissions
Network access is used only to reach the Home Assistant URL you configure.
